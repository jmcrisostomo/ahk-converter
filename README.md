# AHK Converter

String Manipulation for AHK Strings.

```bash
php convert.php
```

## Usage

```php

# results on output folder
foreach(file("this_txt_file.txt", FILE_IGNORE_NEW_LINES) as $line) {
    $arr_candide[] = $line;
}

```
